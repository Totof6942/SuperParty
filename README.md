Installation
-----

Run these two commands to install it:

``` bash
$ wget http://getcomposer.org/composer.phar
$ php composer.phar install --dev --prefer-source
```

Install the database:

``` bash
$ mysql -uproject -pproject123 project < config/schema.sql
```
